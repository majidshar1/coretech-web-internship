# 🚀 MH Innovations - Software House Website
**CoreTech Web Internship - Final Project (Task 11)**

## 📋 Project Overview
This project is a premium, fully responsive landing page for a modern software house named **MH Innovations**. It is built as the final capstone project to demonstrate mastery of frontend technologies, UI/UX principles, and client-side scripting.

## ✨ Key Features
1. **Glassmorphism UI:** Modern sticky navigation bar with a backdrop blur effect.
2. **4 Distinct Sections:** - Interactive Hero Section with gradient typography.
   - Services Grid with hover animations and FontAwesome icons.
   - About Us section highlighting company statistics.
   - Contact Section featuring a sleek inquiry form.
3. **Advanced Form Validation:** Pure JavaScript logic that provides real-time visual feedback (red/green borders) and error messages for invalid inputs (Empty fields, Regex for email, Character limits).
4. **Fully Responsive:** Adapts seamlessly to mobile, tablet, and desktop viewports using CSS Grid and Flexbox.

## 💻 Tech Stack
* **HTML5:** Semantic structure and layout.
* **CSS3:** Custom variables, animations, gradients, and media queries.
* **JavaScript (ES6):** DOM manipulation, event handling, and strict form validation.

## ⚙️ Installation & Usage
1. Clone or download this repository.
2. Navigate to the `task-11-final-project` directory.
3. Open `index.html` in any modern web browser (Google Chrome, Firefox, Edge).
4. *No npm packages or build tools required. It runs instantly.*
