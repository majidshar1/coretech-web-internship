# 10 Essential Git Commands

Here are 10 important Git commands and their explanations:

1. **`git init`**: Initializes a brand new local Git repository in your folder.
2. **`git clone <repo-link>`**: Downloads a repository from a remote server (like GitHub) to your local machine.
3. **`git status`**: Shows the current state of your working directory (which files are changed or untracked).
4. **`git add .`**: Adds all modified and new files to the staging area, preparing them for a commit.
5. **`git commit -m "message"`**: Saves the staged changes into your local repository with a short descriptive message.
6. **`git push origin main`**: Uploads your local commits to the remote GitHub repository.
7. **`git pull`**: Fetches the newest updates from the remote repository and merges them into your local files.
8. **`git branch`**: Lists all local branches in your current repository.
9. **`git checkout -b <branch-name>`**: Creates a new branch and switches your workspace to it immediately.
10. **`git merge <branch-name>`**: Combines the changes from a specified branch into your currently active branch.
